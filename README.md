### directory structure

**./bin**
> facebook常用命令,例如：start,stop.

**./docs*
> facbokk文档

**./public**
> fackbook静态文件,例如：css,js,img,tpl...  

**./routes**
> fackbook路由

**./views**
> facebook视图文件  

### solution

基于NodeJS，采用Mongodb,Stylus,jade,require,socket.io,backbone等技术是实现一个社交网站。
前端自动化工具采用Fetool，一个优秀的前端工具，集成前端优化，自动化工具，代码部署和代码规范于一身。
